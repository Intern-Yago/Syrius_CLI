Documentação

Syrius possui configurações iniciais para páginas web mais simples e mais completas, arquivos git e programas em react

Para criação de arquivos com nomes específicos basta colocar como segundo argumento o diretório com o nome do arquivo ou apenas o nome do arquivo

Para criar na mesma página basta colocar um "." e será adicionado na pasta atual. (É aceito caminho relativo ou absoluto).

A pasta default é a src/, mas com pequenas diferenças q serão esclarecidas nos tópicos

React

O modo react do syrius serve para criação de página ou componente indo diretamente para as respectivas pastas:

"src/page/index.jsx" - para páginas

"src/components/inde.jsx" - ára componentes

Este modo já cria automaticamente páginas de estilização no modo de css modules e formata para ter o link no arquivo inicial

Obs.: Ele faz detecção de projeto React ou ReactNative

Modo de usar: syrius react <page/component> <diretório>

Demonstração dos comandos

syrius react page userList src/page/userList.jsx

syrius react component form .
